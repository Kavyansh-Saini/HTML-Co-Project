## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Kavyansh-Saini/HTML-Co-Project
   cd culinary-corner
   ```

2. **Navigate to the project directory::**

    ```bash
    cd HTML-Co-Project
   ```

3. **Install dependencies:**

   ```bash
   npm i ejs
   ```
4. **Run the server:**

   ```bash
   nodemon server.js
   ```
